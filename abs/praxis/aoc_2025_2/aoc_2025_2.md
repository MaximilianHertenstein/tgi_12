```{.java .cb-run}
import static java.lang.IO.println;
import java.io.IOException;
import java.nio.file.Files;
import java.nio.file.Path;
import java.util.ArrayList;
import java.util.Arrays;
import java.util.Collections;
import java.util.List;

public class Utils {



    static List<String> readCommaSeparated(String fileName) throws IOException {
        var path = Path.of(fileName);
        String content = Files.readString(path);
        String[] splittedFile = content.replace("\n", "").split(",");
        return Arrays.asList(splittedFile);
    }


    public static boolean isInvalidID(long i) {
        var idAsString = String.valueOf(i);
        var splitIndex = idAsString.length() / 2;
        return idAsString.substring(0,  splitIndex).equals(idAsString.substring(splitIndex));
    }



    public static long addInvalidIDsInRange(long start, long end) {
        long acc = 0;

        for(var i = start; i <= end; i++ ) {
            if (isInvalidID(i)) {
                acc = acc + i;
            }
        }
        return acc;
    }

    public static long addInvalidIDsinLines(List<String> lines) {
        long acc = 0;
        for (var line: lines){
            var parts = line.split("-");
            var start = Long.parseLong(parts[0]);
            var end = Long.parseLong(parts[1]);
            acc = acc + (addInvalidIDsInRange(start,end));
        }

        return (acc);
    }



    public static boolean isInvalidID2(long id) {
        var idAsString = String.valueOf(id);
        var splitIndex = idAsString.length() / 2;
        for (int i = 1; i <= splitIndex; i++) {
            if (idAsString.substring(0,  i).repeat((idAsString.length()) / i).equals(idAsString)) {
                return true;
            }
        }
        return false;
    }
}



public class Solutions {

    public static void solution2_x(String fileName) throws IOException {
        var lines = Utils.readCommaSeparated(fileName);
        println(Utils.addInvalidIDsinLines(lines));
    }

}
```

# Hinweise

Mit `Path.of` kannst du ein Objekt der Klasse `Path` erstellen. Dieses steht für den Pfad zu einer Datei.

```{.java .cb-nb line_numbers=false}
var path = Path.of("example_input_2.txt");
```

Der Methode `Files.readString` wird ein Objekt der Klasse `path` übergeben. Sie gibt den Inhalt der Datei als `String` zurück.

```{.java .cb-nb line_numbers=false}
String content = Files.readString(path);
content;
```

Mit der String-Methode `replace` können Teilstring durch andere Teilstring ersetzt werden.
```{.java .cb-nb line_numbers=false}
"hello".replace("l", "lol");
```

Die String-Methode `split` teilt einen String an einem Zeichen auf. Die einzelnen Strings werden in einem Array zurückgegeben.
```{.java .cb-nb line_numbers=false}
var splitted = "a~b~c~d".split("~");
splitted
```

Mit  `Arrays.asList` kann ein `Array` in eine Liste umgewandelt werden.

```{.java .cb-nb line_numbers=false}
Arrays.asList(splitted);
```

# Aufgabe – Kommagetrennte Dateien einlesen

Ergänze in der Klasse `Utils` die statische Methode `readCommeSeparated(String fileName)`.

Die Methode soll:  

- den Inhalt einer Textdatei einlesen  
- Zeilenumbrüche entfernen  
- den Inhalt anhand von Kommas trennen  
- alle Werte in einer `List<String>` zurückgeben  

  
\tiny
```{.java .cb-nb line_numbers=false}
Utils.readCommaSeparated("example_input_2.txt");
```
\normalsize


In den folgenen Methoden musst du mit `long` statt `int` arbeiten, weil die Zahlen, die verarbeitet werden, sehr groß sind.
